# 🎓 初中生超级详细教程 - DOCX 转 SQL 工具

> 这个教程会**一步一步**教你如何使用工具，不需要任何编程知识！

---

## 📚 第一部分：理解这个工具是干什么的

### 简单来说：
这个工具就像一个**翻译器**：
- **输入**：你的 Word 文档（.docx 文件），里面写着考试题目
- **输出**：SQL 文件（.sql 文件），可以导入到数据库

**类比**：就像把中文翻译成英文一样，这个工具把 Word 文档翻译成数据库能看懂的语言。

---

## 🚀 第二部分：开始使用（超级详细步骤）

### 步骤 1️⃣：找到工具文件

1. 打开你的电脑
2. 在桌面上找到 `vet_1` 文件夹
3. 双击打开 `vet_1` 文件夹
4. 找到里面的 `database` 文件夹
5. 双击打开 `database` 文件夹

**你应该能看到很多文件，其中有一个叫 `run_tool.bat`**

---

### 步骤 2️⃣：运行工具（最简单的方法）

#### 方法一：双击运行（强烈推荐！）

1. **找到文件**：在 `database` 文件夹中找到 `run_tool.bat` 文件
   - 它的图标看起来像一个齿轮或者小窗口
   - 文件名是：`run_tool.bat`

2. **双击它**：用鼠标左键快速点击两下这个文件

3. **会发生什么**：
   - 会弹出一个黑色的窗口（不要害怕！）
   - 黑窗口会显示一些文字，正在检查你的电脑
   - 如果缺少需要的程序，它会自动帮你安装
   - 等待几秒到几十秒（取决于你的网络速度）

4. **成功标志**：
   - 黑窗口会自动消失（或者保持在后面）
   - 会弹出一个彩色的窗口，标题是 "DOCX 转 SQL 工具 - 兽医考试题库"
   - **这就是工具的主界面！**

---

### 步骤 3️⃣：如果双击没反应怎么办？

#### 方法二：使用命令提示符（稍微复杂一点）

**第一步：打开命令提示符**

1. 在 `database` 文件夹的空白处，按住键盘上的 **Shift 键**（就是那个向上的箭头 ⬆️）
2. 同时点击鼠标**右键**
3. 会弹出一个菜单
4. 找到并点击 **"在此处打开 PowerShell 窗口"** 或 **"在此处打开命令窗口"**
5. 会弹出一个蓝色或黑色的窗口

**第二步：输入命令**

在这个蓝色/黑色窗口里：

1. 输入（或复制粘贴）以下内容：
   ```
   python docx_to_sql_tool.py
   ```

2. 按下键盘上的 **回车键（Enter）**

3. 等待工具界面弹出

**如果提示错误：**
可能是缺少依赖库，先输入这个命令：
```
pip install python-docx
```
按回车，等待安装完成，然后再试一次运行工具。

## 🎨 第三部分：使用工具界面

当工具成功打开后，你会看到一个窗口，分成几个部分：

### 界面说明（从上到下）

```
┌─────────────────────────────────────┐
│   DOCX 转 SQL 工具                   │  ← 标题
│   将兽医考试题目DOCX文件转换为SQL格式  │
├─────────────────────────────────────┤
│ 📁 文件选择                          │
│   输入文件: [____________] [浏览...] │  ← 选择你的 Word 文档
│   输出文件: [____________] [浏览...] │  ← 选择保存位置
├─────────────────────────────────────┤
│ ⚙️ 设置                              │
│   默认科目: [临床兽医学 ▼]           │  ← 选择科目
├─────────────────────────────────────┤
│ [开始转换] [清空] [退出]             │  ← 操作按钮
├─────────────────────────────────────┤
│ 📄 SQL 预览                          │
│                                     │
│  (这里会显示生成的 SQL 代码)          │  ← 预览区域
│                                     │
└─────────────────────────────────────┘
```

---

### 步骤 4️⃣：选择输入文件（你的 Word 文档）

1. **点击第一个"浏览..."按钮**（在"输入文件"那一行的右边）

2. **会弹出文件选择窗口**，就像你平时打开文件一样

3. **找到你的 Word 文档**：
   - 例如：`disease.docx`（这是示例文件，在 database 文件夹里）
   - 或者你自己准备的题目文档

4. **点击文件，然后点击"打开"按钮**

5. **结果**：输入文件那一栏会显示你选择的文件路径

---

### 步骤 5️⃣：选择输出位置（保存到哪里）

1. **看输出文件那一栏**：工具已经自动帮你填好了
   - 它会在同一个文件夹里，自动生成一个 `.sql` 文件
   - 例如：如果你选的是 `disease.docx`，输出文件会是 `disease.sql`

2. **如果你想改名字或换位置**：
   - 点击第二个"浏览..."按钮
   - 选择你想保存的位置
   - 输入你想要的文件名
   - 点击"保存"

3. **建议**：用默认的就好，不用改！

---

### 步骤 6️⃣：选择默认科目

1. **找到"默认科目"下拉菜单**

2. **点击它**，会出现 4 个选项：
   - 基础兽医学
   - 预防兽医学
   - 临床兽医学
   - 综合应用

3. **选一个**：
   - 如果你的题目文档里每道题都写了科目，这个设置不影响
   - 如果有些题目没写科目，会用这个默认科目

4. **建议**：选"临床兽医学"（这是最常用的）

---

### 步骤 7️⃣：开始转换！

1. **检查一遍**：
   - ✅ 输入文件已选择
   - ✅ 输出文件路径正确
   - ✅ 默认科目已选择

2. **点击"开始转换"按钮**（绿色的大按钮）

3. **等待过程**：
   - 窗口底部会显示进度：
     - "正在读取文件..."
     - "正在解析题目..."
     - "正在生成SQL..."
   
4. **完成！**
   - 会弹出一个对话框，显示：
     ```
     转换完成!
     
     ✅ 成功: XX 道题目
     ⚠️  失败: XX 道题目
     ```
   
5. **点击"确定"**

---

### 步骤 8️⃣：查看结果

1. **在预览区域**（窗口下半部分），你可以看到生成的 SQL 代码

2. **SQL 文件已保存**：
   - 到你选择的输出位置（或默认位置）
   - 找到生成的 `.sql` 文件

3. **如果想重新转换**：
   - 点击"清空"按钮
   - 重新选择文件
   - 再次点击"开始转换"

4. **退出工具**：
   - 点击"退出"按钮
   - 或者直接关闭窗口（点右上角的 ❌）

## 📝 第四部分：准备你的 Word 文档

### Word 文档应该怎么写？

你的 Word 文档（.docx）应该按照这个格式写题目：

```
题目1：
题干：犬瘟热病毒主要侵害的系统是
A. 消化系统
B. 呼吸系统和神经系统
C. 泌尿系统
D. 生殖系统
答案：B
解析：犬瘟热病毒主要侵害呼吸系统和神经系统。
科目：预防兽医学

题目2：
题干：以下哪些症状是典型表现？（多选）
A. 高热
B. 呕吐
C. 腹泻
D. 白细胞显著减少
答案：ABCD
解析：典型症状包括这四个。
科目：临床兽医学
```

### 重要规则（必须遵守！）

#### ✅ 必须要有的部分：
1. **题干**：题目的问题
2. **选项**：A、B、C、D...（至少 2 个）
3. **答案**：用字母表示
4. **解析**：答案的解释

#### 📌 可以不写的部分：
- **科目**：如果不写，会用你在工具里选的默认科目
- **题号**：写不写都行（"题目1"、"第1题"、"1."都可以）

#### 🔴 答案格式：
- **单选题**：只写一个字母，例如 `答案：B`
- **多选题**：写多个字母，例如 `答案：ABCD` 或 `答案：ABD`

#### 💡 科目可以简写：
工具会自动识别并转换：
- 写"临床" → 自动变成"临床兽医学"
- 写"预防" → 自动变成"预防兽医学"
- 写"基础" → 自动变成"基础兽医学"
- 写"综合" → 自动变成"综合应用"

## 🔧 第五部分：常见问题解答

### ❓ Q1: 双击 run_tool.bat 没反应怎么办？

**可能原因1**：电脑没安装 Python

**解决方法**：
1. 打开浏览器
2. 访问 https://www.python.org/downloads/
3. 下载并安装 Python（选择最新版本）
4. **重要**：安装时要勾选 "Add Python to PATH"
5. 安装完成后重启电脑
6. 再试一次双击 run_tool.bat

**可能原因2**：杀毒软件拦截了

**解决方法**：
1. 暂时关闭杀毒软件
2. 或者将 `database` 文件夹添加到信任列表
3. 再试一次

---

### ❓ Q2: 提示"需要安装 python-docx 库"

**解决方法**：
1. 不用担心！run_tool.bat 会自动帮你安装
2. 等待安装完成（可能需要几十秒）
3. 如果自动安装失败，手动安装：
   - 按照"方法二"打开命令窗口
   - 输入：`pip install python-docx`
   - 按回车
   - 等待完成

---

### ❓ Q3: 转换后提示"失败 XX 道题目"

**原因**：有些题目格式不对，工具无法识别

**解决方法**：
1. 看弹出的错误提示，会告诉你哪里有问题
2. 常见问题：
   - 缺少答案
   - 缺少选项
   - 答案字母超出范围（例如只有A-D，答案却写E）
3. 修改 Word 文档中对应的题目
4. 重新转换

---

### ❓ Q4: 生成的 SQL 文件怎么用？

**两种方法**：

**方法A**：直接查看内容
1. 用记事本或其他文本编辑器打开 `.sql` 文件
2. 可以看到里面的 SQL 代码
3. 但是**不建议**用 Windows 自带记事本，可能会乱码

**方法B**：导入到数据库（这才是最终目的）
1. 这个需要用到 Supabase 或其他数据库
2. 如果你不知道怎么做，可以把 `.sql` 文件交给懂技术的人
3. 或者参考其他教程（这部分比较复杂）

---

### ❓ Q5: 我想一次转换很多题目怎么办？

**答案**：把所有题目都写在同一个 Word 文档里！

**步骤**：
1. 创建一个新的 Word 文档
2. 按照格式，一道一道地写题目
3. 题目之间用空行分隔
4. 保存文档
5. 用工具转换这个文档

**例如**：你可以在一个文档里写 100 道题，工具会一次性全部转换！
