# 🔍 科目名称检查 SQL

请在 Supabase SQL Editor 运行这个查询：

```sql
-- 查看数据库中有哪些科目名称
SELECT subject, COUNT(*) as count 
FROM vet_exam_questions 
WHERE is_real_exam = true
GROUP BY subject
ORDER BY subject;
```

这会显示数据库中实际使用的科目名称。

---

## 前端发送的科目名称

前端代码发送的是：
- `基础兽医学`
- `预防兽医学`
- `临床兽医学`
- `综合应用`

如果数据库中的科目名称不一样（例如只是 `基础`、`预防`、`临床`），就会导致查询返回 0 条结果。
