# 🎯 找到问题了！环境变量格式错误

## ❌ 问题

你的 Netlify 环境变量的 **Value（值）** 包含了变量名！

**当前错误的值**：
```
SUPABASE_URL=https://pyavvzsgnusbspcskvpz.supabase.co
```

这导致代码读取到的是 `"SUPABASE_URL=https://..."` 这个字符串，而不是 URL！

---

## ✅ 修复步骤（5分钟）

### 步骤1：编辑 SUPABASE_URL

1. 在 Netlify 环境变量页面
2. 点击 `SUPABASE_URL` 右侧的 **编辑图标** 或 "Options" → "Edit"
3. 在 **Value** 字段中，**删除前面的 `SUPABASE_URL=` 部分**
4. 只保留：
   ```
   https://pyavvzsgnusbspcskvpz.supabase.co
   ```
5. 点击 **Save**

## 📝 格式说明

Netlify 环境变量设置时：

**Key（变量名）**：填写变量名，例如 `SUPABASE_URL`  
**Value（值）**：只填写值本身，不包含变量名和等号

### 示例

| Key | Value |
|-----|-------|
| `SUPABASE_URL` | `https://pyavvzsgnusbspcskvpz.supabase.co` |
| `SUPABASE_ANON_KEY` | `eyJhbGciOiJI...` |

❌ **不要这样**：
| Key | Value |
|-----|-------|
| `SUPABASE_URL` | `SUPABASE_URL=https://...` |
